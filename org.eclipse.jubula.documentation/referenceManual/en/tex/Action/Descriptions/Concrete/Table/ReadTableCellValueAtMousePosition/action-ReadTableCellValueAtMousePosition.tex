% CAP description for Table --> Store Value at Mouse Position

\begin{itemize}
\item Use this action to read the value in the cell where the mouse is currently placed so that you can use the value in the cell as data for other \gdsteps{}.
 \item This can be used in conjunction with the action  \bxcaption{select cell} with a click count of 0.  
\end{itemize}
\bxtipp{This action is unsupported for the HTML toolkit}
