% CAP description for Table --> Context Menu by Textpath (Specified Cell) --> Column
\begin{itemize}
\item Use this parameter to enter the column number which contains the cell whose context-sensitive menu you want to activate.
\item The first column is \bxshell{1} and so on.
\item Headings do not count as a column.
\item If you want to activate the context menu for  a cell which is in the fourth column, enter \bxshell{4}. 

\end{itemize}
