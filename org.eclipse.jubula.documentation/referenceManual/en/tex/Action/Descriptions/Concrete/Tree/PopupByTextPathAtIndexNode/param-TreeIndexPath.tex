% CAP description for Tree --> Context Menu by Textpath (Node specified by indexpath) --> Tree Indexpath
\begin{itemize}
\item Use this parameter to specify the path to the node on which you want to open the context menu.
\item Enter the path as an indexpath.
\item Make sure you give the whole path (start from the very beginning of the tree).
\item The first node is always \bxshell{1}. 
\item Use slash {\tt '/'} to separate parent nodes from child nodes, e.g. \bxshell{1/2}
\end{itemize}
