% CAP description for Tree --> Context Menu by Textpath (Node specified by indexpath) --> Context Menu Textpath
 \begin{itemize}
\item Use this parameter to specify the menupath in the context menu to the item you want to select.
\item Enter the menupath as a textpath.
\item Make sure you give the whole menupath (start from the very beginning of the menu). 
\item Use slash {\tt '/'} to separate menu entries, e.g. \bxshell{File/Open}
\end{itemize}
