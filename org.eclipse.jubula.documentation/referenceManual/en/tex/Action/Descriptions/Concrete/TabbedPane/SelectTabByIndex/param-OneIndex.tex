% CAP description for Tabbed Pane --> Select Tab by Index --> Index

\begin{itemize}
\item Use this parameter to specify which tab you want to select/activate. 
\item Refer to the tab using its index.
\item The first (leftmost) tab has an index of \bxshell{1}, the second \bxshell{2} and so on.
\end{itemize}
