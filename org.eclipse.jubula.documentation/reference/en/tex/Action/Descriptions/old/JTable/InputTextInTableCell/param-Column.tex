% CAP description for Table --> Input Text (Specify Cell) --> Column
\begin{itemize}
\item Use this parameter to specify the column which contains the cell.
\item The column is given by the column number.
\item The first column is \bxshell{1} and so on.
\item Headings do not count as a column.
\item If you want to enter text into a cell which is in the fourth column, enter \bxshell{4}. 
\end{itemize}
