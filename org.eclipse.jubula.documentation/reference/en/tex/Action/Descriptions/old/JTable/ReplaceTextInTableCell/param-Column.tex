% CAP description for Table --> Replace Text (Specify Cell) --> Column
\begin{itemize}
\item Use this parameter to enter the column number which contains the cell whose text you want to replace.
\item The first column is \bxshell{1} and so on.
\item Headings do not count as a column.
\item If you want to replace text in a cell which is in the fourth column, enter \bxshell{4}. 

\end{itemize}
