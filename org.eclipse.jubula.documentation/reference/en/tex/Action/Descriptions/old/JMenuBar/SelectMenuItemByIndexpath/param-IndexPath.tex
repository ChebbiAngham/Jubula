% CAP description for Menu Bar --> Select Menu Item by Indices --> Indexpath
 
\begin{itemize}
\item Use this parameter to specify the menupath to the item you want to select.
\item Enter the menupath as an indexpath.
\item For example, if you want to close the application, and the ''close'' menu is the seventh option in the first menu, then enter \bxshell{1/7} as a parameter.
\item Make sure you give the whole menupath (start from the very beginning of the menu).
\item The first menu item is always \bxshell{1}. 
\item Use slash {\tt '/'} to separate menu entries.
\end{itemize}
