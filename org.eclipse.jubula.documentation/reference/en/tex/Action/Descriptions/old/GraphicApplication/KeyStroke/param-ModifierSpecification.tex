% CAP description for Application --> Key Combination --> Modifier Keys
\begin{itemize}
\item Use this parameter to specify a key to be pressed at the same time as the main key. 
\item You can use this parameter to specify one of the following:
\begin{itemize}
\item \bxshell{shift}
\item \bxshell{ctrl}
\item \bxshell{meta}
\item \bxshell{alt}
\item \bxshell{ctrl shift}
\item \bxshell{ctrl alt}
\end{itemize}
\item You can also select \bxcaption{none} if you do not need a modifier.


\end{itemize}
