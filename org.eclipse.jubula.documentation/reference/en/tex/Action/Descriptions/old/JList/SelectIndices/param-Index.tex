% CAP description for List --> Select Indices --> Index/Indices
\begin{itemize}
\item Use this parameter to enter the item or items you want to select.
\item Use numbers separated by commas to indicate which items you want to select.
\item For example, if you want to select the first, fourth and sixth items from the list, enter: \bxshell{1,4,6}.
\end{itemize}
