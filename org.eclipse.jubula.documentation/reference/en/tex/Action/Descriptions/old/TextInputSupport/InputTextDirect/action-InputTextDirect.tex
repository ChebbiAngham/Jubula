% CAP description for Component with Text Input --> Input Text
\begin{itemize}
\item Use this action to enter text into a component.
\item The text you want to enter is given as a parameter.
\item \jb{} realizes this by:
\begin{itemize}
\item Clicking once on the component if it is not already ready to support keyboard input.
\item Entering the text at the current cursor position.
\end{itemize}
\item To delete any text already in the component, use the action\bxcaption{replace text}.
\item To enter text before/after previous text, see \bxcaption{Insert Text Before/After Pattern} in the \bxcaption{Text Field} component.
\end{itemize}
