% common description for Operator parameter

%\begin{itemize}
%\item The \bxname{operator} parameter has four possible values. 
%\item \bxcaption{not equals} looks for something that does \emph{not exactly} match.
%\item  \bxcaption{equals} looks for an \emph{exact} match.
%\item Select \bxcaption{simple match} to use a simple match expression \bxpref{simplematch}. 
%\item Select \bxcaption{matches} to use a regular expression \bxpref{regex}. 
%\item Regardless of the Operator used: If there is more than one match, the first found will be chosen. 
%\end{itemize}
