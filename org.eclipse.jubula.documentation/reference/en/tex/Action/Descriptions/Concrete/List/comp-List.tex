% CAP description for List
A \bxname{List} is a component which displays a
number of values. Lists allow single or  multiple values to be selected. 

Because the comma (,) is a special symbol for lists, if you want to use a comma as part of your parameter value, you have to mask it. See the section later in this document \bxpref{specialchar} for more details.
