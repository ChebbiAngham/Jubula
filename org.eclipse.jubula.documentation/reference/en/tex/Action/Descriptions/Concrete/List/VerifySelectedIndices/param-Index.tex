% CAP description for List --> Verify Selected Indices --> Index/Indices
\begin{itemize}
\item Enter the index or indices for the items you want to check.
\item For example, if you want to check that the first and third values are selected, enter \bxshell{1,3}.
\item The first item is always \bxshell{1}. 
\item Separate items using commas {\tt ','}
\end{itemize}
