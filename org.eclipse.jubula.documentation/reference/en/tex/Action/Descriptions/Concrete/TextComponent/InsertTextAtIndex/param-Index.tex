% CAP description for Text Field/Text Area/Editor Pane/Text Pane --> Insert Text after Index --> Index/Indices
\begin{itemize}
\item Enter the index after which the text will be inserted.
\item The index is given as the number of characters up to the point where you want to insert your text.
\item The first character is \bxshell{1}.
\item Spaces and symbols also count as  characters.
\item For example, if you want to enter a text after the string \bxshell{My example}, enter the index \bxshell{10}.
\end{itemize}
