% CAP description for Table --> Select Value from Row
\begin{itemize}
\item This action lets you select a cell in a table.
\item To select the cell, you enter the row number the cell is in and the value contained in the cell you want to select.
\item Only the first cell which matches the criteria will be selected.
\item This is achieved by clicking on the specified cell.
\end{itemize}

\bxtipp{This action is unsupported for the HTML toolkit}
