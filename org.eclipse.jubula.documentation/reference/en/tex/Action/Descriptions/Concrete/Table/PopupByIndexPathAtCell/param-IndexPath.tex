% CAP description for Table --> Context Menu by Indexpath (Specified Cell) --> Indexpath
  \begin{itemize}
\item Use this parameter to specify the menupath in the context menu to the item you want to select.
\item Enter the menupath as an indexpath.
\item Make sure you give the whole menupath (start from the very beginning of the menu).
\item The first menu item is always \bxshell{1}. 
\item Use slash {\tt '/'} to separate menu entries, e.g. \bxshell{1/2}
\end{itemize}
