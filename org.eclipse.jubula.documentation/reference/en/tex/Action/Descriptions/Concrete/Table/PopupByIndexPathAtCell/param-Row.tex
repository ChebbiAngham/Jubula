% CAP description for Table --> Context Menu by Indexpath (Specified Cell) --> Row
\begin{itemize}
\item Use this parameter to specify the row which contains the cell whose context-sensitive menu you want to activate.
\item The row is given by the row number.
\item The first row is \bxshell{1} and so on.
\item The headings do not count as a row. 
\item If you want to activate the context menu for  a cell which is in the second row, enter \bxshell{2}. 
\end{itemize}
