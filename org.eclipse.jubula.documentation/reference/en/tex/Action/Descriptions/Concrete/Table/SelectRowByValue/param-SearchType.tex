% CAP description for JTable --> Select Row By Value --> Search Type
% 
\begin{itemize}
\item The \bxname{search type} parameter has two options. 
\item  \bxcaption{absolute} begins searching for the value at the first entry.
\item Select \bxcaption{relative} to begin the search for the value at the currently selected entry. This allows you to search for the next occurence of a value, ignoring all elements before the selected entry. 
\end{itemize}
