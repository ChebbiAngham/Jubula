% CAP description for Tree --> Store Text at Mouse Position

\begin{itemize}
\item Use this action to read the value on the node where the mouse is currently placed so that you can use the value of the node as data for other \gdsteps{}.
 \item This can be used in conjunction with the action  \bxcaption{select node} with a click count of 0.  
\end{itemize}
