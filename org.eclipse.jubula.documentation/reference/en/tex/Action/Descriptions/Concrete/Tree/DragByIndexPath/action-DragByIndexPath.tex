% CAP description for Tree --> Drag Node by Indexpath

\begin{itemize}
\item This action drags the node you specify.
\item You give the path to the node to be dragged using an indexpath.
\item \app{} realizes this by:
\begin{itemize}
\item Moving the cursor over the component.
\item If the tree is not expanded, \app{} expands it.
\item Holding the mouse button you specify, and the item is dragged. 
\end{itemize}
\item You \textbf{must} follow this action with a \bxname{drop} action
\end{itemize}
