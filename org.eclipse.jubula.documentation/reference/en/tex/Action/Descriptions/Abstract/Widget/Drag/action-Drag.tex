% CAP description for Graphics Component --> Drag


\begin{itemize}
\item This action drags the component you specify.
\item The cursor is moved over the component.
\item The mouse button you specify is held, and the component is dragged. 
\item You \textbf{must} follow this action with a \bxname{drop} action
\end{itemize}

\bxtipp{This action is unsupported for the HTML toolkit}
