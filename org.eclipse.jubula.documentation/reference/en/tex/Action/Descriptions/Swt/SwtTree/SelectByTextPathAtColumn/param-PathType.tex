% CAP description for Tree Table --> Select by Textpath (Specify Column) --> Path Type

\begin{itemize}
\item The \bxname{path type} parameter has two values. 
\item  \bxcaption{absolute} begins searching for the path at the root of the tree. Use this value when you enter the full path.
\item Select \bxcaption{relative} to begin the search for the path at the currently selected node. This allows you to enter a partial path, ignoring all elements above the selected node. 
\end{itemize}
