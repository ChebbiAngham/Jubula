This section provides a reference of all currently supported \gdaut
components and the actions that can be applied to them. 

There are two types of actions that can be applied to a given component: \bxname{execute} and \bxname{check}. 
\bxname{Execute} actions are marked with an 'e' in the description,
whereas \bxname{check} actions are marked with a 'c'.
