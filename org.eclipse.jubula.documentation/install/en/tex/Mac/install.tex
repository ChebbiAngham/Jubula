
\begin{enumerate}
\item Browse to the directory where you saved the setup file you downloaded. 
\item Double-click on the setup.dmg icon. The setup directory will open. This directory must later be ejected from the desktop. 
\item Double-click on the \bxcaption{GD Installer} icon. 
\item From this point, follow the instructions for the Unix installation \bxpref{Unix}. 
\end{enumerate}

 

\bxwarn{The Mac OSX Mountain Lion gatekeeper may prevent the install4j generated installer from starting.  This is caused by a too-restrictive setting on the Mountain Lion gatekeeper security feature. Setting the security level to Mac OSX Lion compatible level resolves the issue.}
