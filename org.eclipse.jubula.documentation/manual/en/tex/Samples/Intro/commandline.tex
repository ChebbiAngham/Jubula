When using the Adder \gdaut{}, the following command line parameters are 
available to change the appearance of the application:

\begin{description}
\item[\bxshell{-c}]{This produces an extended version of the Adder. There are 
three modes, two of which have deliberate errors. The Blackjack Mode delivers an error for the calculation \bxshell{17+4} and the Deep Thought Mode produces an error when calculating \bxshell{40+2}. } 
\item [\bxshell{-a}]{This produces a different implementation of the Adder, 
using combo boxes and a label instead of text fields.}
\item [\bxshell{-s}]{This produces a version of the Adder with a delay when the \bxmenu{Help}{about}{} menu is selected. The dialog appears after 5 seconds. }
\end{description}
