This category contains one \gdcase{}. The \gdcase{} contains the same \gdcase{} as in the previous example \bxpref{sampleslibrary}. 

The difference here is that the data is not entered in the \gddatasetsview{}, but is held in an Excel file. 


This \gdcase{} is reused in the executable \gdcase{} \bxname{1.3\_SIMPLE\_ADDER\_TEST\_WITH\_EXCEL\_DATA}, which is nested in the \gdsuite{} of the same name. 


In order to run this test, you will need to take the following steps:
\begin{enumerate}
\item Open:\\
\bxmenu{Window}{Preferences}{}
\item Select the \bxname{\jb{} preferences} from the left-hand side.
\item In the \bxname{path to data files and local scripts} field, change the path to:\\
\bxname{YourGUIdancerInstallationDirectory/aut}
\item This is the root directory for external files. 
\item In the \gdpropview{}, the path to the Excel file is given as a relative path to this directory:\\
\bxname{./Data.xls}
\end{enumerate}
