\app{} supports testing of RAP \gdauts{}. To be able to test RAP applications, select the \bxname{RAP} toolkit in the \gdproject{} properties. 

When you select RAP as the \gdproject{} toolkit, the library \gdproject{} \bxname{unbound\_modules\_concrete} are automatically reused in your \gdproject{}. The actions in these libraries are described in the reference manual \bxextref{\gdrefman}{ref,actparam}.
