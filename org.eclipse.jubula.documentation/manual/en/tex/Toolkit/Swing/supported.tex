\app{} supports \gdauts{} written with the Swing GUI toolkit according to the following points:


\begin{itemize}
\item The \gdaut{} is written using Java 1.4 or higher. 
\bxtipp{Swing \gdauts{} written in Java 1.4 cannot be started using an executable file \bxpref{configuringaut} and cannot be started using the \bxname{autrun} option \bxpref{autrun}. For both of these option,s Java 1.5 or higher is required.}
\item It uses a Sun JRE.
\bxwarn{\gdauts{} written in NetBeans or which are based on the NetBeans framework are currently not supported.}
\end{itemize}
