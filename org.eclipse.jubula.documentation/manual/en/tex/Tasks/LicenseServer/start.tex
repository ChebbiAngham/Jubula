If the license server  is not running as a service on the network, you will need to start it before you can start \gd{}. 

\begin{enumerate}
\item In the \gd{} installation directory, navigate to the \bxname{licenseserver} directory.
\item Select \bxname{startlicenseserver.exe} in Windows and enter GDLicenseserver in Linux. 
\item The license server will start. 
\bxtipp{It takes 5 minutes for the license server to start up. }
\end{enumerate}
