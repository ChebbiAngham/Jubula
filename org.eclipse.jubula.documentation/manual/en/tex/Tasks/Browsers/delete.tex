\begin{enumerate}
\item To delete a \gdcase{}, \gdsuite{}, \gdjob{} or component name, select it in the browser.
\item Select \bxcaption{delete} from the context-sensitive menu. 
\item You can also delete items by pressing \bxkey{DELETE}
\end{enumerate}

\bxwarn{If the item you want to delete is referenced somewhere in your test, you must first remove it from the place where it is reused before deleting it.}
