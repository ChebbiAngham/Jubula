Once a \dash{} server has been started \bxpref{TasksDashCustom}, then you can start the web application in the browser. 

Enter the following URL:\\
\begin{verbatim}
http://HOSTNAME:PORTNUMBER/dashboard
\end{verbatim}

Once the application has loaded, you will see the \reportpersp{} and can use it to view and analyze test results \bxpref{TasksDashUse}.
