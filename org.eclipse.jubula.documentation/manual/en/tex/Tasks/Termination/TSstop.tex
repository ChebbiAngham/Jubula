\index{Stop!Test Suite}

When a \gdsuite has finished, it stops automatically. 

To stop a \gdsuite before it has finished, select the
 \bxcaption{Stop \gdsuite} button 
\gdmarpar{../../../share/PS/stop}{ stop \gdsuite} 
on the toolbar.
