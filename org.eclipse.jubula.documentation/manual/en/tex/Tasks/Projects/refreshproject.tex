\index{Project!Refresh}
\index{Refresh Project}

\begin{enumerate}
\item To refresh a \gdproject{}, first select the \gdproject{} in the \gdtestsuitebrowser{}.
\item Select:\\
\bxmenu{File}{Refresh}{}.

This reloads the data for the \gdproject{} from the database, ensuring in a multi-user environment that all \gdproject{} data is current.

\bxtipp{If a \gdproject{} which your current \gdproject{} reuses has been changed, then you must refresh the current \gdproject{} for the changes to take effect. }

\end{enumerate}
