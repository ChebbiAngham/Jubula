You can find the places where you have used a central test data set from within the \gddataeditor{}:

\begin{enumerate}
\item Select the central data set you want to search for and select:\\
\bxmenu{Show where used}{}{}\\
from the context-sensitive menu. 
You can also press \bxkey{F7} to show the places where the data set has been used.
\item All the places where you have used this data set in this \gdproject{} will be shown in the \bxcaption{Search Result View} \bxpref{TasksSearchResultView}.
\item If you want to change the \gdcases{} that use this central test data set so that they use a different column from the data set, then you can start this action from the \gdsearchresultview{} \bxpref{TasksChangeCTDSColumn}.
\end{enumerate}
