\index{objectmapping}
The GTK graphics toolkit (often used with Linux) can make object mapping with the default \gd{} settings impossible for certain components. If the \gdaut{} that you are mapping is running under Linux, please change the object mapping shortcut key combination to anything \emph{other than} \bxkey{Ctrl+Shift+[0-9 or A-F]}.
