\index{rdesktop}
\begin{itemize}
\item Under a very specific configuration involving \emph{rdesktop 1.5}, \app{} is known to frequently fail tests with an action error: 'Timeout received before confirming the posted event'. 
\item This occurs when \emph{rdesktop 1.5} is used to log into a user account used for the \ite{} as well as a separate account for the \gdagent{}. 
\item The problem does not occur under \emph{rdesktop 1.4.1}.
\end{itemize}
