\index{Command Line Client}

If you are using the default embedded \gddb{}, you will need to enter the following information as command line parameters:
\begin{description}
\item[-dbscheme]{Embedded}
\item[-dbuser]{sa}
\item[-dbpw]{<empty>}

\end{description}
If you have changed the name of the default \gddb{} scheme in the configuration tool, you should alter the parameter for the \bxname{-dbscheme} accordingly.
