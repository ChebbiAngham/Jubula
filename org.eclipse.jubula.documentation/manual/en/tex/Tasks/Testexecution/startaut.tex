\index{AUT!Start}
\index{Start!AUT}
To start the \gdaut{}, you must:
\begin{itemize}
\item have defined an \gdaut{} \bxpref{Defineaut}
\item have connected to the \gdagent{} \bxpref{TasksAgentConnect}
\item if you want to start your \gdaut{} from the \ite{}, and not using the \bxname{autrun} option, then you must have configured an \gdaut{} \bxpref{configuringaut}
\end{itemize}

\bxtipp{You can only start \gdauts{} from the \ite{}  which support the current working language. Change the working language if necessary by selecting a language from the list next to the language button on the toolbar.}
\begin{enumerate}
\item On the toolbar, click on the arrow next to the \bxcaption{Start \gdaut{}} button. 
\gdmarpar{../../../share/PS/startAUT}{ start \gdaut}
\item Select an \gdaut{} and the configuration to start it with from the list.
\item The \gdaut{} will be started in the current working language. 
 \item Clicking directly on the \bxcaption{Start \gdaut{}} button will start the most recently started \gdaut{} (marked with a tick in the list of \gdauts{} and configurations). 
 \end{enumerate}

Once the \gdaut{} has been started, it will appear in the running \gdauts{} view.
\bxtipp{Further information on starting specific types of \gdaut{} is available in the toolkit-specific details \bxpref{Toolkit}.}
