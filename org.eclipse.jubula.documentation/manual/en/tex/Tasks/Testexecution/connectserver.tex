\index{Connect!AUT Agent}
\index{AUT Agent!Connect}
\label{connectserv}
To be able to connect to the \gdagent{} you must first start the \gdagent  \bxpref{otherportnumber}.


\begin{enumerate}
\item On the toolbar, click on the arrow next to the \bxcaption{Connect to \gdagent} button. 
\gdmarpar{../../../share/PS/startServer}{ connect to \gdagent}
\item Select an \gdagent host and port number to connect to from the list. 
\item Clicking directly on the \bxcaption{Connect to \gdagent} button will start the most recently started \gdagent (marked with a tick in the list of \gdagent{} hosts and port numbers). 
\end{enumerate}
