\index{Test!Execution!Prepare}
To be able to start a test you must:

\begin{enumerate}
\item Create a \gdsuite{} \bxpref{TSeditor}.
\item Add \gdcases{} to be executed to the \gdsuite{} \bxpref{TasksEditorAdd}
\item Define an \gdaut{} \bxpref{Defineaut}.
\item Configure an \gdaut{} \bxpref{configuringaut} if you want to start the \gdaut{} from the \ite{}.  
\item Choose an \gdaut{} for the \gdsuite{} \bxpref{confsuite}.
\item Set the working language to the language you want to execute the test in.
\item Connect to the \gdagent{} \bxpref{TasksAgentConnect}.
\item Start the \gdaut{} \bxpref{startaut} (either from the \ite{} or using the \bxname{autrun} command). 
\item Complete object mapping \bxpref{objectmappingtasks}.
\item Make sure that your test data is complete in your \gdcases{}.
\end{enumerate}


\bxtipp{It is a good idea to set your preferences for test results in the preferences dialog \bxpref{gdprefs} before test execution.}
