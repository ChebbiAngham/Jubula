\index{autrun command!Define AUT}
\index{Define AUT!autrun command}

The \bxname{autrun} command can be used as an alternative to starting your \gdaut{} from the \ite{}. 
(i.e. with an \gdaut{} configuration \bxpref{configuringaut}). 
It can only be used if your \gdaut{} is written in Java 1.5 or above, and you are using a Java Standard Edition JRE. 


\bxwarn{The \bxname{autrun} command cannot be used for HTML or pure SWT \gdauts{}.}

The command allows you to start your \gdaut{} independently, on a machine where the \gdagent{} is running. The  \ite{}, when connected to this \gdagent{} will then recognize the running \gdaut{} as a testable application. 

To use the \bxname{autrun} command:
\begin{enumerate}
\item Ensure that the \gdagent{} is installed on the machine where you will be starting the \gdaut{}. 
\item Navigate to the \bxname{server} directory in the installation via the command line.
\item Start your \gdaut{} via the command line by entering \bxshell{autrun.exe} under Windows or \bxshell{autrun} under Linux then the following parameters: 
\begin{table}[h]
\label{autrunparams}
	\centering
	\begin{tabular}{|l|l|}

	\hline
	\textbf{Detail}&\textbf{Parameter}%&\textbf{Example}
\\
		\hline
                -h 
                &\bxshell{-h}\\
                & Gives parameter help\\

                  & \bxshell{-w <directory>}\\
		  &Enter the working directory for the \gdaut{}\\
                  \hline

\bxtipp{If your \gdaut{} is an RCP \gdaut{}, use \bxshell{-data'<WORKSPACE>'} after the executable file to specify the workspace the \gdaut{} should use.}
\subsubsection{Creating an \gdaut{} definition from a running \gdaut{}}
\label{createAUTDef}
Once you have started an \gdaut{} using the \bxname{autrun} command, you can automatically generate an \gdaut{} definition \bxpref{Defineaut} for this \gdaut{}:

\begin{itemize}
\item In the \gdrunautview{}, select the \gdaut{} you want to define (it will be marked as an unknown \gdaut{} ID).
\item Select:\\
 \bxmenu{Create AUT Definition}{}{}\\
from the context menu.
\item The \gdaut{} definition window will appear. Complete the dialog \bxpref{Defineaut} and click \bxcaption{OK}.
\end{itemize}
