There are a few symbols with a special function (\bxfigref{specialchars}). If you want to use these symbols as a part of a text entry or verification, you must mask them with a backslash. 


\subsection{Verbatim text symbol}
\label{verbatimsymbol}
If you have a lot of characters that need to be masked, you can place the data in single quote marks:\\
'DATA'\\
 This means that any special functions of the characters between the quote marks are ignored.

\subsection{Symbols with special meanings for certain parameters}
In addition to these general symbols, there are a few symbols which have a special meaning for particular components (\bxfigref{specialcompsymbols}). They must be masked if you want to use them without their special function \bxname{for these components only}. Masking these symbols must be done with a double backslash.
