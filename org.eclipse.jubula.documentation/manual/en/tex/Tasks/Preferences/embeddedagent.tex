You can find the preference page for the embedded \gdagent{} under:\\
\bxname{Test-\gdagent{}-Embedded \gdagent{}}\\
in the preferences dialog.

On this page, you can enter the port number that the embedded \gdagent{} \bxpref{TasksAgentConnect} should use when you connect to it \bxpref{TasksAgentConnect}. The default is 60000.
