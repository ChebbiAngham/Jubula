\subsection{Windows Users}
\begin{enumerate}
\item Start \GD{} from the start menu:\\
\bxmenu{Start}{\GD{}}{\GD{}}.

\item You can also start \gd{} using a command line argument:\\ 
(\bxshell{guidancer.exe}). 
\end{enumerate}

\subsection{Unix Users}
Enter \bxshell{guidancer} into the shell. 

\subsection{Choosing a workspace}
\begin{enumerate}
\item When you start \gd{}, a dialog appears asking you to choose a workspace. 
\bxtipp{The workspace is where your preferences are saved.}
\item Select the default workspace offered, or locate new one. 
 \item Select \bxcaption{OK}.
 \end{enumerate}
The default workspace was configured during installation, and can be changed in the configuration tool. You can find details on the configuration tool in the installation manual. 

\subsubsection{Deleting the list of used workspaces}
To remove workspaces from the drop-down box in the workspaces dialog:
\begin{enumerate}
\item Go to: \\
\verb+%USERPROFILE%\user\com.bredexsw.guidancer.ui.ide\  +
\item In this folder, there is a file called \bxname{recentWorkspaces.xml} where the list of used workspaces can be deleted. 
\end{enumerate}
