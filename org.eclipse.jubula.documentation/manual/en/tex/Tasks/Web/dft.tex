Although \app{} does not require you to name components in your \gdaut{}, you neverthless make your \gdaut{} more testable by doing so. In HTML \gdauts{}, you can enter a name for your components in the id tag.  %% with a unique name for your components. You can then enter this tag in the \gdaut{} configuration \bxpref{autconfigbasic} so that it is recognized as the component name for object recognition.
