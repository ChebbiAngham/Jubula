\index{Delete!Test Case}
\begin{enumerate}
\item To delete a \gdcase{}, select it in the \gdtestcasebrowser{}.
\item Select \bxcaption{delete} from the context-sensitive menu. 
\item You can also delete \gdcases{} from the \gdtestcasebrowser{} by pressing \bxkey{DELETE}
\end{enumerate}

\bxwarn{If the \gdcase{} you want to delete is used in another \gdcase{} or in a \gdsuite{}, you must first remove it from the place where it is reused before deleting it.}
