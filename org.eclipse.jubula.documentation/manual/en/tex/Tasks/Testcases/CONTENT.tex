\gdcases{} are the ''building blocks'' (keywords) in \app{}. They can: 
\begin{itemize}

\item contain other (referenced) \gdcases{} \bxpref{TasksEditorAdd}
\item contain \gdsteps{} \bxpref{specsteps}
\item be used in \gdsuites{} \bxpref{TasksEditorAdd}
\item be used as \gdehandlers{} \bxpref{customizedehandler}
\end{itemize}
