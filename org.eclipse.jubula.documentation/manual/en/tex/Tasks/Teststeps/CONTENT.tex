\bxtipp{Instead of creating \gdsteps{}, we recommend using the library of \gdcases{} \bxpref{usingtemplate}. Each \gdcase{} in the library contains one \gdstep{}, with all the parameters already referenced. Using the library makes test creation much easier!}
