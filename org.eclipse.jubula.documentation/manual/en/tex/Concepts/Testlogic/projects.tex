\index{Project}

\gd{} organizes tests into \gdprojects{}. 
A \gdproject{} is the top-level container in \gd{}. \gdprojects{} contain \gdsuites{} and the details about the \gdauts{} to be tested by these \gdsuites{} (how to start them, in which language etc.). 

You can work jointly on \gdprojects{} using the multi-user \gddb{}, and you can also share \gdprojects{} using the import and export functions.
