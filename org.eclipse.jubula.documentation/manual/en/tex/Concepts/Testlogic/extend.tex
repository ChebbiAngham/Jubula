\index{Extensibility}
\app{} is extensible in two ways:
\begin{itemize}
\item New actions can be defined for existing components
\item New project-specific components which are derived from standard components can be added and actions for them created.
\end{itemize}

Both of these processes require somebody to  program the new action or 
component, and to link the new action or component to the \gdagent 
process. 

For information on how to do this, see the separate guide to enhancing \app{}.
