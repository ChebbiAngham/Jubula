\index{Client}
\index{Server}

\jb{}{} is a client-server application:  
\begin{description}
\item[The \gdclient{}]{is where tests are specified and evaluated.}
\item [The server (\gdserver){}]{controls the application under test (\gdaut{}) and executes the specified tests. }
\end{description}

 You can install the client and \gdserver on different machines; even on different platforms. This lets you specify and run  your tests on different machines, and means that you can run the same test on different platforms.
