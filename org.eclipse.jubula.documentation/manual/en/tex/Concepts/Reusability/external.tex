\index{External Data}
\index{Excel File}

\gd{} also lets you use data from an external source (an Excel file). You can give the path to an Excel file containing data for your test in the \gdpropview{}. You can use an Excel file for data if you have used references in the \gdsteps{} in a \gdcase{}. 

You can enter the path to the Excel file in the \gdpropview{} for the \gdtestcaseeditor{} or for the \gdtestsuiteeditor{}. You can also overwrite the path to the Excel file when you reuse a \gdcase{}.
