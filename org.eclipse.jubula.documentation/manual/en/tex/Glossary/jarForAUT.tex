\item[JAR]{
\label{jar}
The \bxname{Java}\texttrademark  Archive file format. It allows multiple classes and other program files and data to be combined into one compressed file, which often represents a complete program.
In this context, a JAR describes the \bxname{Java}\texttrademark
  Archive of the \gdaut.
}
