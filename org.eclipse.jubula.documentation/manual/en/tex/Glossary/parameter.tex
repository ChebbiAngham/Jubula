%\index{Parameter}
\item[Parameters]{
The information that is given to an action to perform a \gdstep{}, such as
 name, value, etc.
A parameter consists of:
\begin{itemize}
\item a name
\item a type 
\item a value
\end{itemize}
Most actions require one or more parameters.
}
