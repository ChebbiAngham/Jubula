%\index{Action}
\item[Action]{
A method which is carried out on a \bxname{Component}.
An \bxname{Action} is part of a \gdstep{}, and serves to either execute a command (e.g. click, enter text), or verify property or state information about a \bxname{Component} (e.g. check \bxname{Component} name, check visibility).
}
