\item[AUT Configuration]{
The information required in order to locate and start an \gdaut{} in the configuration desired by the user.
An \bxname{AUT Configuration} includes program information such as \bxname{JRE} location and class path, and \gdaut configuration information such as command-line parameters and environment variables. Details  about the \gdaut are given when a project is created, and are entered into the relevant
fields in the \gdsuiteeditor{}.  
}
