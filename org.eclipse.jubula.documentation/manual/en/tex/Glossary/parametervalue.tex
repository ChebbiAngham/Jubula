%\index{Parameter!Value}
\item[Parameter Value]{
The contents of the parameter which is sent to an action. A parameter value can either be a static value (e.g. \bxshell{Hello}, 123), a reference (e.g. \bxshell{=REFERENCE}) or a variable.
}
