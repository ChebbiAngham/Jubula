%\index{Environment}
%\gdhelpid{environmentForAutContextId}{Environment}
\item[Environment]{
The surroundings in which the \gdaut runs, as defined in the \bxname{AUT Configuration} in environment variables, e.g. ''LANG=en$\_$UK'' may be used to tell the \gdaut that the language environment is English.
}
