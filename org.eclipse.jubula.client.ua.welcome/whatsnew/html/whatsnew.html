What's new:

Toolkit updates

  • We have added new actions for the GEF toolkit to address connection points on figures and to check figure properties.
  • The GEF inspector also provides property information about the selected figure
  • It is now possible to address GEF figures using the Jubula Client API. It is also possible to write extensions for use in the Jubula Client API.
  • We have added various new actions to the JavaFX toolkit, including actions on tree tables, actions to check tooltips, actions on date pickers, and various other additions.
  • In the JavaFX object mapping mode, you can now choose to collect all supported parent components as well as the highlighted component
  • In the SWT/RCP toolkit, there are new actions to check the property at mouse position for tables, trees and tree tables. You can also check JFace tooltips.

Other updates

  • We have drastically improved the performance for opening test result reports in the ITE
  • The order of data sets in the Data Sets View can now be changed using buttons to move rows up or down

The list above is just an overview. For a more complete guide to new features, read the release notes.
