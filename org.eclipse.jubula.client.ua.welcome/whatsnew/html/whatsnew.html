Back-up!
See the migration section for information on updating to a new version of GUIdancer.

What's new:

  1. You can now use Mylyn in GUIdancer to manage tasks and contexts. This makes working with larger projects more comfortable.
  2. You can create central test data sets and reuse them in your Test Cases to avoid having to write the same data more than once.
  3. GUIdancer has a new search function, which lets you search for keywords and test data throughout the whole project. From the Search Result View, you can easily spring to each search result to view or edit it.
  4. We have introduced Manual Test Steps which can be defined and executed manually, so that all your tests are in one GUIdancer project.
  5. GUIdancer can now test applications that were started by other applications in the test.
  6. Test results from the Test Result Summary View can be exported as HTML and XML reports.
  7. We have some new actions (check existence of window, store value on tree node and table cell).
  8. We've added a new view to the Object Mapping Editor. This Split View provides a better overview and finer control of your Object Mapping.

The release notes provide more information on new features and known issues. You can access the release notes from the installation directory, in the documentation folder.
