What's new:

  • When entering data, you can now also enter mathematical and date functions. You can use the functions provided or add your own.
  • Test result reports now contain details about the data used at each level as well as the duration of each Test Step, Test Case and Test Suite.
  • When data are missing in editors, this is shown directly in the editor.
  • Categories are now supported in the Test Suite Browser and Central Data Sets Editor.
  • There is a new analysis framework with some example metrics to provide information on the project.
  • A new option "Save As New Test Case" lets you select existing Test Case references from within an editor and save them into a new Test Case.

The list above is just an overview. For a more complete guide to new features, read the release notes.
