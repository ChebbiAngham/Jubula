What's new:

  • We've added a new layer in the ITE that allows you to write tests via an offical Java client API
  • You can now update fields in external ALM tools based on the results of a test run
  • The testexec tool has new options including no-run, optional result and data parameters as well as the ability to name the result files
  • We have made many updates to the JavaFX support
  • Version numbers can now consist of major, minor and micro versions
  • There is a new description view for adding longer information to nodes in your test

The list above is just an overview. For a more complete guide to new features, read the release notes.
