What's new:

ITE updates

  • It is now possible to copy and paste items (Test Cases, Event Handlers, Test Steps, ...) between editors.
  • You can now add comments as nodes within the editors.
  • There is a new action "invoke method" for the Java toolkits that lets you call methods offered by the AUT from within your test.
  • It is now possible to adapt the Object Mapping profile for selected technical names.
  • Jubula now supports XLSX Excel files as an external data source.
  • A new view which shows the logs from the execute external command actions has been introduced.

Performance updates

  • We have improved the performance for working with ITE: Checks for completeness, problems and teststyle are now performed in the background and no longer block working with the ITE

The list above is just an overview. For a more complete guide to new features, read the release notes.
