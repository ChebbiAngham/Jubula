What's new:

  • There is a new key combination in object mapping mode. With that it is now possible to map the current component at mouse position and all its supported parents from the AUT's hierarchy in the JavaFX toolkit.
  • You can now check the property of items in tables, trees, lists and tree tables .
  • You can now use ub_tbl_checkPropertyAtMousePosition and ub_tre_checkPropertyAtMousePosition in the SWT toolkit.
  • We now use Selenium version 2.46.0.
  • We added an example for our Client API using JavaFX.
  • It is now possible to change the order of single data sets in the Data Sets View.
  • Performance for opening Test Result Details in the Reporting Perspective has been improved.

The list above is just an overview. For a more complete guide to new features, read the release notes.
