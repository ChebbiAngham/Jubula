What's new:

  • You can now test Eclipse e4 applications, with and without the compatibility layer.
  • HTML test result reports now include screenshots taken when an error occurs. You can click the screenshot to increase and decrease its size.
  • The properties view is more compact and easier to recognize when it is not editable.
  • Navigation between Browsers and Editors with double-clicks is easier
  • Search scopes are supported in the search view.
  • There are new functions available for accessing node attributes and cells in central test data sets
  • There are new actions for preparing an AUT for shutdown and synchronizing its restart (experimental in Kepler)

The list above is just an overview. For a more complete guide to new features, read the release notes.
