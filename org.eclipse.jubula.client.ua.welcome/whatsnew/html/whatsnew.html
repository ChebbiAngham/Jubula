What's new:

ITE updates

  • The New Project Wizard has been greatly reduced to only require the most necessary information.
  • You can choose to create a new project based on a pre-defined template to give you some hints about how to structure your tests
  • The error messages when AUT starts fail have been improved
  • You can now create simple extensions for non-supported components via the ITE, without having to write a full extension.
  • In JavaFX AUTs, you can select text that is actually a shape and can also test shapes as graphics components.
  • Sliders are now supported in all Java toolkits
  • There are various improvements and new features for using code coverage.
  • The datasets view can now be filtered

    • The list above is just an overview. For a more complete guide to new features, read the release notes.
