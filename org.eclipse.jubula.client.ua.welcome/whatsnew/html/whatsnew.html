What's new:

ITE updates

  • Added the If-Then-Else node to have conditional test execution
  • Added loop nodes like the While-Do-node which are executing the same test cases depending on a condition
  • Component names types are calculated differently
  • A timeout parameter has been added to most of the check actions which lets you wait for a specific status
  • It is now possible to skip specific test steps if a parameter value has the value ##jbskip##

    • The list above is just an overview. For a more complete guide to new features, read the release notes.
