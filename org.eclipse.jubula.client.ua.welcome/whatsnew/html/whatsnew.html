What's new:

  • Embedded AUT Agent for working with a local Agent on your machine.

  • HTML Test Result Reports can be expanded again.
  • Copy ID to clipboard now also works for Test Suites.

The list above is just an overview. For a more complete guide to new features, see the Jubula website New and Noteworthy section or read the release notes.
