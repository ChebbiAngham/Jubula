Overview

There is a lot of information on the web about the tool: See the Jubula website and the general FAQs for more details.

The following points give you an overview of the documentation.


  1. The user manual contains information on the concepts, how to complete various tasks with the tool, and information on the installed example projects.
  2. The reference manual contains information on the supported components and actions, and on regular expressions, language codes, shortcuts etc.
  3. The installation manual guides you through the installation.
  4. Read the release notes to keep track of the changes from version to version.
  5. The extension manual gives directions and examples on adding your own components and actions.

You can access the documentation from the installation directory in PDF format. You can also access the user manual and reference manual from the Help menu, and as context-sensitive help.
