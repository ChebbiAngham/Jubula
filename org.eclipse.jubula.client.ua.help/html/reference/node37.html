7. Overview of Components

The following diagram shows which abstract components Jubula uses and for which actions they can be used for.

Figure 7.1: Jubula component structure

Figure 7.1 provides an overview of Jubula 's concrete and abstract component structure. This figure can be interpreted as follows: the arrows show inheritance relations between the components. Thus, a Label component has no actions of its own, but inherits them all from Component with Text. The Label component also inherits any actions that Component with Text also inherited. In this case, these are the actions from Graphics component.
