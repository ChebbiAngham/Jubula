11. Keyboard layout files

For SWT and RCP AUT's , you need to define a keyboard layouts for the AUT (i.e. which keyboard layout is set for the system on which the AUT runs).

Jubula already has the keyboard layouts for German (Germany) and English (US) saved. If you want to use another keyboard layout, follow the instructions in the next section to create one.
