4.5 Testing RAP AUT's

Jubula supports testing of RAP AUT's . To be able to test RAP applications, select the RAP toolkit in the Project properties.

When you select RAP as the Project toolkit, the library Project unbound_modules_concrete are automatically reused in your Project . The actions in these libraries are described in the reference manual -->Reference Manual .
