3.8.5 Editing Test Cases

From the Test Case Editor , you can edit a Test Case . You can:

  • add other Test Cases 3.7.2.
  • add and edit Test Steps 3.13.1.
  • rename the Test Case 3.6.1 or the Test Cases it contains 3.7.4.
  • add a comment to the Test Case 3.7.5.
  • add, edit or translate data for the Test Case 3.9.10.
  • extract Test Cases from within this Test Case to make reusable modules 3.7.7.
  • deactivate specific Test Cases in this Test Case so they are ignored during execution 3.7.6.
