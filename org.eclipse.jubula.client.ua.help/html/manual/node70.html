3.5.1 Two options to start applications under test (AUT's )

Jubula offers two options to start your AUT for testing:

Via an AUT configuration:
This option means that you create an AUT configuration in your Project , and the AUT is started by Jubula 3.5.3.
Using the autrun command:
This option lets you start an AUT without creating a configuration. Certain start parameters are required for the AUT so that Jubula can locate it 3.5.4.
