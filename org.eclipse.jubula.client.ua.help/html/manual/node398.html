4.6 Testing Windows Store App AUT's

You can write tests for Windows Store Apps AUT's that are designed for the modernUI desktop when you select the toolkit concrete, win or winapps in the Project properties. The win, winapps and concrete toolkits actually contain the same actions as each other.

When you select winapps as the Project toolkit, the library Project unbound_modules_concrete is automatically reused in your Project . The actions in this library are described in the reference manual -->Reference Manual .
