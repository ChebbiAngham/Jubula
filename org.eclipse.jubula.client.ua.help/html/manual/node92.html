3.7.6.2 AUT configuration for Windows Store AUT's

To be able to start a Windows Store AUT , you should select the winApps toolkit as the AUT toolkit in the AUT definition dialog.

You must enter the following information to be able to start a winApps AUT :

  1. Enter the basic configuration details as described earlier 3.7.2.
  2. Enter the name of the app. This is the name that is shown when the AUT is installed. This name is locale-dependent - you will have to have separate AUT configurations to test your app in different languages.
  3. Enter any necessary command-line arguments for the AUT in the AUT Arguments field.
