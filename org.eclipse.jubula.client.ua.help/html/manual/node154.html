3.14.2 Executing and analyzing manual tests

Once you have created a manual Test Case 3.14.1, you can execute it.

  1. Start the AUT 3.16.3.
  2. Start the Test Suite containing the manual Test Steps 3.16.4.
  3. For each manual Test Step in the test, a dialog will appear with the descriptions and data you entered for the action to perform and the expected behavior.
  4. Execute the Test Step as described, enter a comment if necessary, then click either "passed" or "failed" in the dialog to move to the next Test Step .
    If the Test Step fails, then an automatic screenshot will be taken 3.19.7, and any Event Handlers 3.18 for this Test Case will be activated.
  5. Once the dialog has been closed, the next Test Step dialog will appear.
  6. At the end of the test, you can see the test result report like any other automated test 3.23.
