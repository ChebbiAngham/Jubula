3.23.5 Using the search dialog

You can open the search dialog from the toolbar, by pressing »CTRL+H« or via the menu:
Edit --> Search

The search dialog allows you to search the current Project in the current working language for:

  • Keywords 3.23.5.1 (Test Cases , Test Steps , Test Suites , Test Jobs and categories).
  • Test data used in Test Cases or central test data sets 3.23.5.2.
  • Files in the workspace 3.23.5.3.
