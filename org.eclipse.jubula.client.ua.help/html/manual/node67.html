3.4.10.2 Exporting all of the Projects from the database

  1. Select:
    Test --> Export all
    .
  2. Choose or create a folder where you want the Projects to be stored.
  3. Click "OK".
  4. The Projects (including the test result summaries 3.17.3) will be saved as .xml files.
    When you export all Projects , all versions of each Project are exported.
