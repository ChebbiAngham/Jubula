5.3.1.1 Executable tests

The executable test category contains the following:
The executable use cases
as described in 5.1.3
A subcategory called Event Handlers
which is discussed later 5.4.
Any tests which reproduce errors
from the bug-tracking system, called ticket tests. Each ticket Test Case is named with the ticket number to link it to the bug-tracking system.
