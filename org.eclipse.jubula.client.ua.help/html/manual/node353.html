7.6 Test hierarchy

The following sections describe the test hierarchy in Jubula . Basically, you can think of the test hierarchy as a tree structure which can be of any size, complexity and depth (7.1).

The hierarchical structure of Jubula gives you flexibility in your tests to make changes and keep the test in touch with current software development.
