3.8 Working with Test Cases

Test Cases are the ''building blocks'' (keywords) in Jubula . They can:

  • contain other (referenced) Test Cases 3.7.2
  • contain Test Steps 3.13.1
  • be used in Test Suites 3.7.2
  • be used as Event Handlers 3.18
