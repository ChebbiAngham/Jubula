3.16.2 Connecting to the AUT Agent

To be able to connect to the AUT Agent you must first start the AUT Agent 3.1.

  1. On the toolbar, click on the arrow next to the "Connect to AUT Agent " button.
      connect to AUT Agent

  2. Select an AUT Agent host and port number to connect to from the list.
  3. Clicking directly on the "Connect to AUT Agent " button will start the most recently started AUT Agent (marked with a tick in the list of AUT Agent hosts and port numbers).
