3.16.1 Prerequisites for test execution

To be able to start a test you must:

  1. Create a Test Suite 3.11.1.
  2. Add Test Cases to be executed to the Test Suite 3.7.2
  3. Define an AUT 3.5.2.
  4. Configure an AUT 3.5.3 if you want to start the AUT via Jubula .
  5. Choose an AUT for the Test Suite 3.11.2.
  6. Set the working language to the language you want to execute the test in.
  7. Connect to the AUT Agent 3.16.2.
  8. Start the AUT 3.16.3 (either via Jubula or using the autrun command).
  9. Complete object mapping 3.15.1.
  10. Make sure that your test data is complete in your Test Cases .
It is a good idea to set your preferences for test results in the preferences dialog 3.24 before test execution.
