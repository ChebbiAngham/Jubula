3.17.3.3 Changing the amount of result summaries shown in the Test Result Summary View

In the test result preferences 3.19.8 you can alter the amount of days that a test result summary is shown. The top of the Test Result Summary View shows the amount of days that test result summaries are being shown for. Any unshown summaries are still present in the Database and can be shown by increasing the amount of days.
