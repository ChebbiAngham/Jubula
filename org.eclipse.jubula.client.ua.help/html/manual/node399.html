4.6.1 Supported WinApps AUT's

The following AUT's are supported:

  • You must be using Windows 8 or higher.
  • We have performed our tests with an AUT in .NET version 4.5.
  • In order to perform tests on winapps AUT's , you must have installed the ITE or the AUT Agent in the Program Files directory (if you are testing locally, then the ITE must be installed here, if you are testing remotely, then the AUT Agent must be installed in Program Files).
  • You must also install the ITE or the AUT Agent with the certificate offered during the installation.
