3.3.1 Logging in to the Database

  1. Projects are stored in a Database . Before you can access a Project , you have to log in to the Database .
  2. The log-in dialog for the Database (3.1) appears automatically the first time you need it (e.g. when you try to open or create a Project ).
    Figure 3.1: Database login
    If you are using the default demo (embedded) Database , you will automatically be logged into the Database .
  3. Select the Database you want to use and enter your username and password.
    If you previously had another Jubula version installed, you may see the information that your current Database version is not compatible with the latest version. If this is the case, then you can automatically migrate your Database 3.3.3.
