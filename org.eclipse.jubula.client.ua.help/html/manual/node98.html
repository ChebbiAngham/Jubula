3.8.4.1 Copying the ID of a Test Case to the clipboard

  1. Select the Test Case whose ID you want to use from the Test Case Browser .
    Make sure you select the original specification of the Test Case , not a place where it has been reused. Press »F6« to find the original specification of a selected Test Case .
  2. Select:
    Copy ID to clipboard

    from the context-sensitive menu.
  3. You can now paste the ID into your external system.
