3.8.2 Creating tests from the library of pre-defined Test Cases

Everything you need to create tests is already available for you in each Project . The library of Test Cases which appears automatically in each new Project contains all the actions supported by Jubula as well as some examples of more complex keywords. For general information about the library, read the section later on 3.8.2.2. To learn how to use the library to create tests, read the next section 3.8.2.1.
