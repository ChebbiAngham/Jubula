3.33.2 Using Chronon when testing your AUT

You can configure your AUT to run with Chronon recording so that you can analyze any errors that occur during your automated test runs. See the section later 3.33.2.3 on analysing the reports for information on the tooling required to do this.

Jubula users: The support for Chronon recording in AUT's is currently only available in the standalone software.
