3.8.7 Categories for Test Cases

Categories let you group Test Cases together in the Test Case Browser .

Creating a category:

  1. In the Test Case Browser , select:
    New --> Category
    from the context-sensitive menu.
  2. Name the category when prompted.
  3. You can add Test Cases to this category using drag-and-drop.
  4. You can also nest categories in other categories.
  5. You can do this either by dragging and dropping a category into another category, or by right-clicking in an already present category and choosing the option to create a new category.
    For best practices on how to use categories in the Test Case Browser , see the section on Best Practices 5.3.1

Creating Test Cases in a category:

  1. In the Test Case Browser , select the category in which you want to create the Test Case with a single-click.
  2. Create a Test Case via the context-sensitive menu 3.8.1.
  3. You can also simply double-click on the category in which you want to create the Test Case .
