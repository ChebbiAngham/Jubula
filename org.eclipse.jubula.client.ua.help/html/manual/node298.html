5.3.2 Best practices for structuring the Object Mapping Editor

Organizing the Object Mapping Editor into categories (5.2) makes it easier to find components you have mapped later on - especially if you have to remap any of them. As a general rule, the categories in the Object Mapping Editor will reflect the categories in the Test Case Browser in the AUT -bound modules category - the AUT should be split up into areas and dialogs, so that you have a simple yet hierarchical overview of your AUT in the Object Mapping Editor 5.3.1.
