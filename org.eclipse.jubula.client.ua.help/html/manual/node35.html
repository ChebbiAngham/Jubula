3.1.1 Windows users

  1. Start the AUT Agent via the start menu:
    Start --> Jubula --> Start AUT Agent
    .
  2. The AUT Agent is started on port number 60000 unless you enter a different port number as a parameter 3.1.3.

    You can see and stop the AUT Agent in the system tray.
