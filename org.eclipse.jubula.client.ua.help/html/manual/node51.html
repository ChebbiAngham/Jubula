3.4.2 Editing the Project and AUT properties

You can open the Project properties dialog (3.3) via:
Test --> Properties

Figure 3.3: Project Properties Dialog

The Project properties dialog lets you see and, in some cases, edit information about:

  1. the Project in general 3.4.2.1.
  2. the Project languages 3.4.2.3.
  3. the AUT's 3.4.2.4.
  4. the Projects that you have reused in this Project 3.5.
