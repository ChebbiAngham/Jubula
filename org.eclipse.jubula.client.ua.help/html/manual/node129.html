3.9.10 Using the Data Sets View to enter data loops and to translate data

The Data Sets View lets you do three things:

  • Enter multiple data sets for a parameter from a Test Case 3.9.10.1.
  • Enter data sets for a central test data set 3.9.8.4.
  • Translate your parameter values into other languages 3.9.10.2.

You can also create central test data sets for your Project to reuse in Test Cases 3.9.8.
