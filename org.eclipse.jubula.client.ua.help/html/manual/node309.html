6.1.3 The Functional Test Reporting Perspective

In the Functional Test Reporting Perspective , you can see an overview of all the test runs in the current Database , for all Projects it contains. You can reopen test runs if the details for the report are still in the Database 3.17.3.1 and analyze the test using the Properties View and the Image View .

The Functional Test Reporting Perspective (6.3) contains the following views:

  • The Test Result View
  • The Properties View
  • The Image View
  • The Test Result Summary View
