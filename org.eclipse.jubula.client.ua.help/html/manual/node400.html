4.6.2 Information on the support for WinApps AUT's

AUT modifications
No modifications are necessary to ensure that the AUT can be tested. The AUT can be deployed normally: no extra steps are needed for a test deployment.

Installation specifics
In order to start modernUI AUT's , you must have installed the BREDEX certificate as a part of the installation, and the installation must be in a folder with exclusive administrator rights (e.g. Program Files). More information on this is in the Installation Manual.
