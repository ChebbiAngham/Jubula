4.1 Testing Swing AUT's

You can write tests for Swing AUT's when you select the toolkit concrete or Swing in the Project properties. The Swing and concrete toolkits actually contain the same actions as each other.

When you select Swing as the Project toolkit, the library Project unbound_modules_concrete is automatically reused in your Project . The actions in this library are described in the reference manual -->Reference Manual .
