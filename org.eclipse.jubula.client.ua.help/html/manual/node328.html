3.33 Using Chronon

Chronon (3.0) is supported as a monitoring agent to record information about the behavior of your AUT during a test.

The information gathered by Chronon can be used to help analyze any errors found in the program.

Jubula Feature users: the Chronon plugins are only available in the standalone version.
