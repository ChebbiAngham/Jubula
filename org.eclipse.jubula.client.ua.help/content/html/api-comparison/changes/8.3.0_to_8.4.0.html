Changes
It is now possible to use the Invoke Method And Store Result command of the Reflection Component from the API by calling the method invokeExternalMethodStoreReturnValue of the Reflection Component. The Result object has been extended by a new getReturnValueAsMap method which returns a Map supplied by the called method.
It is now possible to use the Invoke External Method And Store Result command of the Reflection Component from the API. Just as with the other Store Result commands, one can obtain the result through the getReturnValue and getReturnValueAsMap methods of the Result object.
